Learn Summation 101

What is Summation?

Summation is the process of adding up a series of numbers. The numbers to be added are called addends, or sometimes summands. The result of adding up the numbers is called the sum. The symbol \(\sum\) is used to denote summation. The numbers being summed are called the terms of the sum. The term to be summed is indicated by the index of summation. The index of summation is written below the summation symbol.

Baby Rules of Summation

  • Find the first term of the sequence.
  • Find the last term of the sequence.
  • Find the number of terms in the sequence.
  • Plug the values into the summation formula.

Real Rules of Summation

  • Identify the pattern - Determine if the sequence is arithmetic (constant difference between terms) or geometric (constant ratio between terms) or follows some other pattern.
  • Define the range - Determine the range of values over which the summation is to be performed. This is usually defined by an index variable (e.g., i) and the range (e.g., from i = 1 to n).
  • Write Out the Terms: - Write out the individual terms of the sequence using the index variable. For example, if you have \(\sum_{i}\) from i = 1 to 5, write out the terms as (1, 2, 3, 4, 5).

Some Simple Summation Problems

  1. \(\sum_{i=1}^{5}X_{i}\)

    =

    \(x_{1}+x_{2}+x_{3}+x_{4}+x_{5}\)

  2. \(\sum_{i=1}^{6}X_{i}^{2}\)

    =

    \(x_{1}^{2}+x_{2}^{2}+x_{3}^{2}+x_{4}^{2}+x_{5}^{2}+x_{6}^{2}\)

  3. \(\sum_{i=1}^{3}X_{i} + \sum_{i=5}^{8}X_{i}\)

    =

    \(x_{1}+x_{2}+x_{3}+x_{5}+x_{6}+x_{7}+x_{8}\)

Express each of the following extended sums in the appropriate summation notation:

  1. \(\frac{x_{1}+x_{2}+x_{3}+x_{4}}{n}\)

    =

    \(\frac {\sum_{i=1}^{4}X_{i}} {n}\)

  2. \((3x_{1}+3x_{2}+3x_{3}+3x_{4}) (5x_{1}+5x_{2}+5x_{3}+5x_{4})\)

    =

    \((\sum_{i=1}^{4}3X_{i})(\sum_{i=1}^{4}5X_{i})\)

  3. \((x_{1} + x_{2}+ x_{3} + x_{4})^{2}\)

    =

    \(\left(\sum_{i=1}^{4} X_i\right)^2\)

Reduce each of the following expressions into its simplest form by application of the rules of summation.

  1. \(\sum_{i = 1}^{10}(5X_{i} - 3)\)

    =

    \(5\sum_{i = 1}^{10}X_{i} - 3\sum_{i = 1}^{10}1\)

  2. \((\sum_{i = 1}^{5}X_{i} + 2C) (\sum_{i = 1}^{3}7)\)

    =

    \(\sum_{i = 1}^{5}X_{i} \cdot 21 + 42C\)

  3. \(\sum_{i = 1}^{n}[X_{i} - (\frac{\sum_{i = 1}^{n}X_{i}}{n})]\)

    =

    \(\sum_{i = 1}^{n}(x_{i} - \bar{x})\)

Interpolation vs Extrapolation

Interpolation

Interpolation is the process of estimating a value of a function between two known values. Extrapolation is the process of estimating a value of a function outside the range of known values. Examples? Given data points (x, y) for various x-values, interpolation can be used to estimate the y-value at a specific x-value that falls between the known data points.

Extrapolation

Extrapolation is the process of estimating a value of a function outside the range of known values. Examples? Given data points (x, y) for various x-values, extrapolation can be used to estimate the y-value at a specific x-value that falls outside the range of known data points.
